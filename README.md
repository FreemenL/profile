### 运行环境:

  系统: macOS 10.14.6
  node: v16.17.0
  npm: v8.15.0
  mongodb:  v4.0.9

### 运行步骤:

  1. 安装node和npm
  2. 安装mongodb 导入数据库数据 (./db/test.js)
  3. 安装git
  4. git clone https://github.com/FreemenL/profile.git
  5. 进入项目目录,执行pnpm install安装依赖包
  6. 执行npm run start启动项目
  7. 可部署在blocklet-server中
